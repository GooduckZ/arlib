## Arlib 

Arlib is toolkit for automated reasoning.



## Installation

- Install the requirement packages following `requirements.txt`
    ~~~~
    pip install -r requirements.txt
    ~~~~
- (Optional) Download all binary solvers in the dir `bin_solvers`
